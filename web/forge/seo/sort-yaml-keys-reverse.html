Sort YAML / JSON Keys in Reverse Order — devbench cf --sort-keys-reverse

Sort all keys in a config file in reverse alphabetical order with a single flag. Works across 11 formats — no yq expressions, no jq pipelines.

The Problem: Reverse-Sorting Keys in yq Is Awkward

yq has no sort_keys_reverse() function (see yq#2390). The workaround involves multiple pipes:

# yq: reverse-sort top-level keys only (misses nested keys)
yq '. | to_entries | sort_by(.key) | reverse | from_entries' config.yaml

# jq equivalent (JSON only, also misses nested)
jq '. | to_entries | sort_by(.key) | reverse | from_entries' config.json

Neither approach recurses into nested objects. Both require remembering a multi-step pipeline.

The Solution: devbench cf --sort-keys-reverse

# Reverse-sort all keys (recursively) in a YAML file
devbench cf config.yaml --sort-keys-reverse

# Same for JSON
devbench cf config.json --sort-keys-reverse

# Convert YAML → TOML with keys reverse-sorted
devbench cf config.yaml --to toml --sort-keys-reverse

# Reverse-sort and write back in-place
devbench cf config.yaml --sort-keys-reverse --in-place

--sort-keys-reverse sorts all keys at every nesting level, not just the top level.

Example: Reverse-Sorted Kubernetes Deployment

Input deploy.yaml:

apiVersion: apps/v1
kind: Deployment
metadata:
  name: my-app
  namespace: default
spec:
  replicas: 3
  selector:
    matchLabels:
      app: my-app
devbench cf deploy.yaml --sort-keys-reverse
spec:
  selector:
    matchLabels:
      app: my-app
  replicas: 3
metadata:
  namespace: default
  name: my-app
kind: Deployment
apiVersion: apps/v1

Use Case: Canonical Reverse-Sorted Config for Diffing

# Sort both files the same way before diffing
devbench cf config-v1.yaml --sort-keys-reverse > /tmp/v1-sorted.yaml
devbench cf config-v2.yaml --sort-keys-reverse > /tmp/v2-sorted.yaml
diff /tmp/v1-sorted.yaml /tmp/v2-sorted.yaml

Use Case: Terraform Variables Reverse-Sorted

# Sort terraform.tfvars keys in reverse for review
devbench cf terraform.tfvars --sort-keys-reverse

Versus yq / jq

Task yq jq (JSON only) devbench
Forward-sort keys sort_keys(..) walk(if type == "object" then to_entries | sort_by(.key) | from_entries else . end) --sort-keys
Reverse-sort keys (recursive) ❌ Not natively supported (yq#2390) ❌ Complex multi-step expression --sort-keys-reverse
Works on TOML ❌ ❌ ✅
Works on INI / .env ❌ ❌ ✅
Recursive (all nesting levels) Partial Requires walk() ✅ Always recursive

Pair with --sort-keys for Forward Sort

# Forward sort (Z→A ... A→Z)
devbench cf config.yaml --sort-keys

# Reverse sort (Z→A)
devbench cf config.yaml --sort-keys-reverse

pip install devbench

TSV to JSON Converter — Command Line

Convert tab-separated (TSV) files to JSON, YAML, TOML, or any other supported format in one command. Also works with pipe-separated (|), semicolon-separated (;), and other custom delimiters. Part of DevBench ConfigForge — 100% offline, no data leaves your machine.

pip install devbench

Quick Examples

TSV → JSON

# scores.tsv
# player	score	level
# Alice	100	expert
# Bob	87	intermediate

devbench cf scores.tsv --tsv --to json
# [{"player": "Alice", "score": "100", "level": "expert"}, ...]

# Raw JSON only (pipe-friendly):
devbench cf scores.tsv --tsv --to json --raw

TSV → YAML

devbench cf employees.tsv --tsv --to yaml

TSV → TOML

devbench cf data.tsv --tsv --to toml

Stdin input

cat data.tsv | devbench cf --tsv --from csv --to json --raw

Custom Delimiter (Pipe, Semicolon, etc.)

# Pipe-separated file: id|name|value
devbench cf data.csv --csv-delimiter '|' --to json

# Semicolon-separated (common in European locales):
devbench cf report.csv --csv-delimiter ';' --to yaml

# Explicit tab delimiter (same as --tsv):
devbench cf data.tsv --csv-delimiter $'\t' --to json
Tip: The --tsv flag is a shorthand for --csv-delimiter $'\t'. Both work identically. Use whichever is clearer in your scripts.

TSV → CSV Conversion

# Convert TSV to standard comma-separated CSV:
devbench cf data.tsv --tsv --to csv --raw

# Convert pipe-separated to CSV:
devbench cf report.psv --csv-delimiter '|' --to csv --raw

Batch Conversion

# Convert all TSV files in a directory to JSON:
devbench cf '*.tsv' --tsv --batch --to json --output-dir ./json-out/

# Stream mode for large file collections (memory-efficient):
devbench cf '*.tsv' --tsv --batch --stream --to json --output-dir ./out/

Extract Fields from TSV

# Get value at row index (0-based):
devbench cf data.tsv --tsv --to json --raw | jq '.[0].player'

# Filter rows where score > 90 (pipe to jq):
devbench cf scores.tsv --tsv --to json --raw | jq '[.[] | select(.score | tonumber > 90)]'

# Use --each to extract a single column:
devbench cf data.tsv --tsv --to json --raw | devbench cf --each name --raw

Real-World Use Cases

Database exports

# PostgreSQL COPY exports tab-separated data:
psql -c "COPY users TO STDOUT" | devbench cf --tsv --from csv --to json --raw > users.json

# MySQL tab-separated dump:
mysql -u root -p mydb -e "SELECT * FROM orders" | devbench cf --tsv --from csv --to yaml

Spreadsheet data

# Google Sheets / Excel tab-separated paste:
pbpaste | devbench cf --tsv --from csv --to json --raw

# LibreOffice CSV export (semicolon-separated):
devbench cf export.csv --csv-delimiter ';' --to json

CI/CD pipelines

steps:
  - name: Convert metrics TSV to JSON
    run: |
      devbench cf metrics.tsv --tsv --to json --raw > metrics.json
      cat metrics.json | jq '.[] | select(.status == "FAIL")'

Log file analysis

# Apache access log (space-separated fields):
awk '{print $1"\t"$7"\t"$9}' access.log | devbench cf --tsv --from csv --to json --raw | jq .

# Nginx pipe-delimited custom log format:
devbench cf nginx.log --csv-delimiter '|' --to json --raw | jq '.[] | select(.status == "500")'

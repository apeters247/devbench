Deduplicate YAML / JSON List — devbench cf --unique

devbench cf tags.yaml --unique
Remove duplicate values from any list. Use --unique-by FIELD for object lists.

The Problem: Deduplicating Lists in yq / jq

Removing duplicates from a YAML or JSON list requires remembering expression syntax:

# jq: deduplicate a list of strings
jq '. | unique' tags.json

# jq: deduplicate objects by a field
jq '. | unique_by(.name)' services.json

# yq: deduplicate (yq doesn't have unique_by natively — needs complex expression)
yq '. | map(select(. != null)) | unique_by(.)' tags.yaml

These don't work across formats and the object-dedup syntax is easy to forget.

The Solution: --unique and --unique-by FIELD

# Deduplicate a list of strings
devbench cf tags.yaml --unique

# Deduplicate a list of objects by a field (keep first occurrence)
devbench cf services.yaml --unique-by name

# Navigate then deduplicate
devbench cf log.json --get events --unique

# Deduplicate Kubernetes pods by node name
devbench cf pods.yaml --get items --unique-by spec.nodeName

# Combine: deduplicate, then sort
devbench cf services.yaml --unique-by name --sort-by name

Example: Deduplicate Tags List

Input tags.yaml:

tags:
  - python
  - backend
  - python
  - api
  - backend
  - python
devbench cf tags.yaml --get tags --unique

Output:

- python
- backend
- api

Example: Deduplicate Services by Name

Input services.yaml:

services:
  - name: auth
    port: 8080
    region: us-east
  - name: api
    port: 8081
    region: us-west
  - name: auth
    port: 8080
    region: eu-west
devbench cf services.yaml --get services --unique-by name

Output (first occurrence of each name kept):

- name: auth
  port: 8080
  region: us-east
- name: api
  port: 8081
  region: us-west

Compose with Other List Flags

# Filter, then deduplicate by name
devbench cf pods.yaml --get items --select status.phase=Running --unique-by metadata.name

# Deduplicate log entries then sort by timestamp
devbench cf events.json --get items --unique-by id --sort-by timestamp

# Extract unique tag names from a list of objects
devbench cf releases.yaml --get releases --unique-by tag --each tag

JSON5 Converter — CLI Tool for JSON5 Files

Convert JSON5 files to JSON, YAML, TOML, and 9 other formats from the command line. Auto-detects JSON5 syntax: single quotes, unquoted keys, comments, trailing commas. Pure Python — no Node.js, no npm packages.

What Is JSON5?

JSON5 is a relaxed superset of JSON popular in JavaScript tooling (ESLint configs, Babel configs, VS Code settings). It adds human-friendly features that standard JSON forbids:

  • Single-quoted strings: 'hello'
  • Unquoted object keys: { name: "value" }
  • Comments: // single-line and /* block */
  • Trailing commas: [1, 2, 3,]

Convert JSON5 to JSON

# Convert .json5 file to standard JSON
devbench cf config.json5 --to json

# Auto-detected from content (no --from needed)
devbench cf .eslintrc.js --to json

Input (config.json5):

{
  // Database connection settings
  host: 'localhost',
  port: 5432,
  database: 'myapp',
  options: {
    timeout: 30,
    retries: 3,   // trailing comma is fine
  }
}

Output (--to json):

{
  "host": "localhost",
  "port": 5432,
  "database": "myapp",
  "options": {
    "timeout": 30,
    "retries": 3
  }
}

Convert JSON5 to YAML

devbench cf config.json5 --to yaml
host: localhost
port: 5432
database: myapp
options:
  timeout: 30
  retries: 3

Convert JSON5 to TOML

devbench cf config.json5 --to toml
host = "localhost"
port = 5432
database = "myapp"

[options]
timeout = 30
retries = 3

Auto-Detection

DevBench detects JSON5 automatically from content — no --from flag needed for files with JSON5 syntax:

# Both work:
devbench cf settings.json5 --to yaml
devbench cf settings.json5 --from json5 --to yaml

Works in CI/CD Pipelines

# Extract a value from a JSON5 config in CI
VERSION=$(devbench cf package.json5 --get version)

# Validate a JSON5 config is parseable
devbench cf .eslintrc.json5 --validate && echo "Config OK"

# Convert JSON5 config to YAML for Ansible
devbench cf app.json5 --to yaml > vars/app.yml

No Node.js Required

Most JSON5 tools require Node.js or the json5 npm package. DevBench is pure Python — works in any environment:

pip install devbench
devbench cf config.json5 --to json

No npm install -g json5, no Node version conflicts, no runtime dependencies beyond Python 3.10+.

Install

pip install devbench

ConfigForge — Multi-Format Config File Converter (9 Formats)

Convert between JSON, YAML, TOML, XML, CSV, INI, .env, HCL, and .properties — all 9 formats bidirectionally. Runs 100% offline. Preserves comments. Batch mode included.

⇄

9 Formats Bidirectional

Convert any format to any other: JSON ↔ YAML, TOML ↔ XML, HCL ↔ JSON, .properties ↔ YAML, CSV ↔ INI, and every other pair. Auto-detects input format from content.

#

Comment Preservation

YAML and INI comments are extracted before conversion and re-inserted into the output. No silent data loss — what you document stays documented.

⚡

Batch Mode & CLI

Convert entire directories with cf --batch *.yaml --to json. Zero dependency Python CLI. pip install devbench-cf.

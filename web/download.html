💻 Cross-Platform CLI

One command, works on macOS, Linux, and Windows (via Python):

$ curl -sSL https://naxiai.com/install.sh | bash

Or install via pip: pip install devbench

🚀 Quick Start

$ devbench cf --serve
# Opens the web UI at http://127.0.0.1:8080
$ echo 'hello: world' | devbench cf --to json
# CLI one-liner: YAML → JSON
$ devbench license activate <your-key>
# Activate your license on this machine
$ devbench license verify <your-key>
# Check license status

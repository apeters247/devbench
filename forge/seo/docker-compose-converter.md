# Convert Docker Compose YAML to JSON: A Step-by-Step Guide

Your `docker-compose.yml` is clean YAML, but your deployment tools, CI pipeline, or monitoring system needs JSON. How do you convert a **Docker Compose YAML to JSON** without data loss or security risks?

This guide shows you how using **ConfigForge** (`devbench cf`) — a free, offline CLI that converts between 9 config formats. No uploads, no accounts, no rate limits.

## Why Convert Docker Compose Files?

Three common scenarios:

- **JSON-only orchestration** — Your custom deployment tool or PaaS expects JSON input
- **CI/CD pipeline integration** — You need to transform Compose files for pipeline steps
- **Configuration management** — Feeding Compose data into Ansible, Terraform, or monitoring dashboards

## Basic Conversion

Convert `docker-compose.yml` to JSON with a single command:

```bash
devbench cf --to json < docker-compose.yml
```

Devbench auto-detects the input as YAML and writes clean, valid JSON. The whole operation happens locally — your Compose file never touches a network.

## Handling Multi-Service Docker Compose Files

A typical `docker-compose.yml` with multiple services converts cleanly:

```yaml
# docker-compose.yml (input)
version: "3.8"
services:
  web:
    image: nginx:latest
    ports:
      - "80:80"
  db:
    image: postgres:15
    environment:
      POSTGRES_PASSWORD: secret
```

Output JSON preserves the full service structure including nested fields, arrays, and environment variables:

```bash
devbench cf --to json < docker-compose.yml
# Converts to valid JSON with "web" and "db" service objects
```

## Converting JSON Back to YAML for Docker Compose

Round-trip from JSON back to YAML when your pipeline finishes processing:

```bash
devbench cf --to yaml < processed.json > docker-compose.yml
```

## Batch Converting Multiple Compose Files

If you maintain a monorepo with multiple `docker-compose.yml` files across microservices:

```bash
find . -name "docker-compose.yml" -exec sh -c \
  'devbench cf convert "$1" "${1%.yml}.json"' _ {} \;
```

For tools that support native glob patterns, you can also batch-convert in one invocation:

```bash
devbench cf convert "**/docker-compose.yml" --to json --batch
```

## Converting to Kubernetes Manifests

## Security: Never Upload Compose Files

Docker Compose files contain sensitive information:
- Image names and tags revealing infrastructure patterns
- Service names and internal network topology
- Environment variables with secrets and credentials
- Volume mount paths exposing filesystem structure
- Port mappings and exposed interfaces

**Never paste a docker-compose.yml into an online converter.** ConfigForge runs entirely on your machine — zero external API calls, zero data uploaded, zero telemetry.
